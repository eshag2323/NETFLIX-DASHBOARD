# Project Overview

This project presents an interactive dashboard built using Tableau to analyze Netflix's content library. The dashboard provides insights into movies and TV shows based on genre, release year, ratings, and country distribution.



##  Objectives

* Analyze the distribution of Movies vs TV Shows
* Identify trends in content release over the years
* Explore popular genres and categories
* Understand country-wise content production
* Examine ratings and audience preferences


##  Tools & Technologies Used

* Tableau (Data Visualization)
* Excel / CSV (Dataset)



##  Files Included

* `Netflix_Dashboard.twbx` → Tableau packaged workbook
* `dashboard_preview.png` → Dashboard screenshot



## 🚀 How to Use

1. Download the `.twbx` file
2. Open it in Tableau Desktop
3. Interact with filters and visuals
